## 功能

### 技能冷却追踪
- 实时追踪对面 5 名敌人的召唤师技能冷却（闪现、传送、点燃等）
- 左键点击开始计时，右键点击重置
- 自动计算装备带来的召唤师技能急速（明悟之靴、绯红明悟）
- 通过 Riot API 自动检测敌方符文（星界洞悉 +18 急速）
- Doinb 计时：自动将闪现计时写入剪贴板，游戏内 `Ctrl+V` 直接粘贴到聊天框（如 `15:25Top 20:00Mid`）

### 覆盖层控制
- 按住 `Tab` 显示面板，松开自动隐藏，不遮挡游戏画面
- 始终置顶，全屏游戏中也能正常显示
- 鼠标悬停可交互，离开后自动穿透点击
- `Ctrl+Shift+L` 锁定/解锁窗口位置，防止误拖动

### 其他
- 自动检测游戏状态，进入对局后自动启用，结束后自动关闭
- 最小化到系统托盘，不占用任务栏
- 反应延迟补偿，可在设置中调整
- 窗口位置和设置自动保存

## 使用方法

1. 从 [Releases](https://github.com/YourUser/jjgg-overlay/releases/latest) 下载安装包并安装
2. 启动 JiJiGuGu，等待进入游戏
3. 进入对局后，按住 `Tab` 查看敌方技能面板
4. 看到敌人交闪现时，点击对应技能按钮开始计时
5. 在游戏聊天框中按 `Ctrl+V` 即可发送计时信息给队友

## 快捷键

| 按键 | 功能 |
|------|------|
| `Tab`（按住） | 显示/隐藏覆盖层 |
| `左键点击` 技能 | 开始计时 |
| `右键点击` 技能 | 重置计时 |
| `Ctrl+V` | 游戏内粘贴计时信息 |
| `Ctrl+Shift+L` | 锁定/解锁窗口位置 |
